## Nightmare & Tension System (ADOPTED: Combined Deck v2.0)

### Deck Structure

| Card Type | Count | Function |
|-----------|-------|----------|
| **Minor Cards** | Majority | Small negative effects, dream destabilization |
| **Major Cards** | ~8-12 | Nightmare spawn + escalation effect |
| **Final Card** | 1 | Triggers phase change |

### Game Flow: 3 Phases

| Phase | Deck State | Tension Level |
|-------|------------|---------------|
| **Phase 1** | Full deck | Low threat, building |
| **Phase 2** | Post-reshuffle, trimmed | Escalating, more Major cards |
| **Phase 3** | Highly trimmed | Climactic, Final Surge approaches |

### Key Mechanics

**Shrinking Deck System:**
- After each reshuffle, remove Minor cards
- Major card frequency increases naturally
- Creates organic tension escalation

**Nightmare Spawn Deck (Separate):**
- Provides variety in which nightmares appear
- Not the primary pacing mechanism
- Complements the tension deck

**Board Size:**
- Fixed board creates predictable game window
- Prevents open-endedness
- Game ends when board fills or Final Surge resolves

**Predictable Window:**
- ~60-90 minutes based on board size
- No "stalling" incentive â€” deck advances per turn
- Consistent arc: Incubation â†’ Instability â†’ Climax
